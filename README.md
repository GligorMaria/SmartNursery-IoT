# 🍼 Smart Baby Monitor
 
Real-time temperature & humidity monitoring for your baby's room, powered by a **Raspberry Pi + DHT11 sensor**, **Firebase Realtime Database**, and a **Flutter** mobile app with push notifications.
 
---
 
## Architecture
 
```
DHT11 Sensor
    │
    ▼
Raspberry Pi (tempSensor.py)
    │  writes every 10 s
    ▼
Firebase Realtime Database
    ├── baby_monitor/latest      ← current reading
    └── baby_monitor/readings/   ← full history
    │
    ▼
Flutter App  ←──── Firebase Cloud Messaging (push alerts)
```
 
---
 
## 1 · Firebase Setup
 
1. Go to [console.firebase.google.com](https://console.firebase.google.com) and create a project.
2. Enable **Realtime Database** (start in test mode, secure later).
3. Enable **Cloud Messaging** (FCM) — no extra config needed.
### Raspberry Pi credentials
- Project Settings → Service Accounts → **Generate new private key**
- Save the JSON as `serviceAccountKey.json` next to `tempSensor.py`.
### Flutter credentials
```bash
dart pub global activate flutterfire_cli
firebase login
flutterfire configure          # generates lib/firebase_options.dart
```

## 2 · Raspberry Pi Setup
 
### Wiring the DHT11
 
| DHT11 Pin | Raspberry Pi Pin |
|-----------|-----------------|
| VCC       | Pin 1 (3.3 V)   |
| DATA      | Pin 7 (GPIO 4)  |
| GND       | Pin 6 (GND)     |
 
> Add a **10 kΩ pull-up resistor** between VCC and DATA.
 
### Install dependencies
```bash
pip install Adafruit_DHT firebase-admin
```
 
### Run the sensor script
```bash
python3 tempSensor.py
```
 
Add to crontab for auto-start on boot:
```
@reboot sleep 20 && python3 /home/pi/baby_monitor/tempSensor.py >> /home/pi/baby_monitor/sensor.log 2>&1
```

## 4 · Push Notifications
 
The Python script writes `status: "ALERT"` to Firebase when values are outside safe ranges.  
You can trigger FCM from a **Firebase Cloud Function** (Node.js) that watches the `latest` node:
 
```js
// functions/index.js (Cloud Functions)
const functions = require('firebase-functions');
const admin     = require('firebase-admin');
admin.initializeApp();
 
exports.alertOnBadReading = functions.database
  .ref('/baby_monitor/latest')
  .onWrite(async (change) => {
    const data = change.after.val();
    if (data?.status !== 'ALERT') return;
 
    await admin.messaging().sendToTopic('baby_alerts', {
      notification: {
        title: '⚠️ Baby Room Alert!',
        body:  data.message,
      },
    });
  });
```
 
Deploy with:
```bash
firebase deploy --only functions
```

## 5 · Safe Ranges Used
 
| Parameter   | Min  | Max  | Source |
|-------------|------|------|--------|
| Temperature | 18°C | 22°C | AAP    |
| Humidity    | 40%  | 60%  | AAP    |
 
*(American Academy of Pediatrics recommendations)*

## 6 · DHT11 Sensor Specs (from lab)
 
- Operating Voltage: 3.5 V – 5.5 V
- Operating current: 0.3 mA (measuring) / 60 µA (standby)
- Temperature Range: 0°C – 50°C  |  Accuracy: ±1°C
- Humidity Range: 20% – 90%  |  Accuracy: ±1%
- Resolution: 16-bit for both Temperature and Humidity
